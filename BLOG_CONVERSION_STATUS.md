## 📝 Note Tecniche

### Struttura BlogArticleTemplate:
```tsx
<BlogArticleTemplate
  title="Titolo SEO completo"
  description="Meta description"
  keywords="keyword1, keyword2, keyword3"
  slug="url-slug"
  ogImage="https://giolabriparazioni.it/og-images/slug.jpg"
  author="Giolab Team"
  datePublished="2025-01-XX"
  category="Categoria"
  content={articleContent}
  readingTime={X}
/>
```

### Requisiti per TOC:
- Tutti gli `<h2>` devono avere `id` univoco
- Tutti gli `<h3>` devono avere `id` univoco
- Gli ID devono essere kebab-case (es: `id="come-funziona"`)
- Gli ID devono essere descrittivi del contenuto
